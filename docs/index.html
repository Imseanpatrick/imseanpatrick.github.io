Sloth Selfie Analysis

This is my own project using NLP in practice in my research. I used the Twitter API to scrape over 20,000 Tweets about sloth selfies to examine captive wildlife tourism on Twitter. I labeled these tweets based on four categories and used these Tweets to train a classifier then pseudolabel the remainder of Tweets for analysis.
