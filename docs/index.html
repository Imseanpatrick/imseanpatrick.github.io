Automating Data Validations for Public Dashboards

This arose from my contract with the State of Massachusetts Executive Office of Public Safety and Security, Data Management team. After determining that an intensive weekly validation process was unnecessarily encumbering the workflow of a team member, I worked with the team member to understand the validation process and made a series of Python functions to automatically transform and analyze the data, turning an 8-hour weekly process into a 5-minute weekly process.

Automating Data Validations for Public Dashboards Repository.

Machine Learning for Text Analysis with Python and Introduction to Python Workshops

I created these materials, with assistance and review of some team members, for my Machine Learning for Text Analysis with Python workshop. The goal is to take a novice Python programmer through the evaluation and training of traditional machine learning-based text classifiers. Materials are designed to serve as a resource and template.

Python Workshop Materials Repository.

Twitter API Query Program

This program was created with a former colleague to pull data from the now defunct Twitter API for academic researchers. I acquired all tweets used in my research and analysis using this program. The program iterates over the Twitter API call response to format the JSON to a manageable pandas dataframe.

Twitter API Query Program Repository.

Google File Copier

Google decided to stop offering free storage for academic accounts. As a result of this, Northeastern IT announced the deletion of student-owned Gmail accounts. The NULab folders contained files owned by student accounts that would be lost in the event of account deletion. I made this program on a time crunch to iterate over folders and files in Google Drive and copy them all to a new directory, safe from deletion.

Google File Copier Project Repository.
